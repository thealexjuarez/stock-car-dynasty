# Stock Car Dynasty Project Brief

## Current Build Rules

- Built with Expo React Native and TypeScript.
- Uses Expo Router for navigation.
- Mobile-first dark motorsports-inspired UI.
- Locked bottom navigation: Home, Team, Market, Race, League.
- Current phase is app shell, placeholder screens, shared components, starter data, and clean structure.
- Do not add race simulation logic yet.
- Do not add real authentication yet.
- Do not add monetization yet.
- Do not add backend code yet.
- Do not add save/load logic yet.
- UI must be card-based, readable, and easy to navigate.
- Every screen should answer: what is happening, why does it matter, what can the player do next?
